# Создание таблиц

Для создания таблиц служит команда `CREATE TABLE`.

```
CREATE TABLE table_name
(
   field_name type [constraints],
   field_name type [constraints],
   ...
   field_name type [constraints],
   [constraints],
   [primary_key],
   [foreign_key]
);
```
В квадратных скобках показаны необязательные элементы команды.
