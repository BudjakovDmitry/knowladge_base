# math

Модуль math содержит математические функции

```python
import math

############
# константы
############

math.e  # экспонента
# 2.718....

math.pi  # число pi
# 3.141592653589793

##########
# функции
##########

math.floor(2.567)  # округление в меньшую сторону (до меньшего целого)
# 2

math.floor(-2.567)
# -3

math.trunc(2.567)  # усечение (отбрасывание десятичных цифр)
# 2

math.trunc(-2.567)
# -2

math.sin(2 * math.pi / 180)
# 0.034....

math.sqrt(81)
# 9.0
```
